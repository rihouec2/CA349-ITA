About Fusion Tables Project

CA349 coursework is based on exploiting Fusion Tables to display a large volume of data in a graph or map format.

We use a dataset that is suited to a map based layout and we use Fusion Tables to both display the data and allow for queries to filter the dataset visually.

Our front end is designed for User ease of use and allow the user to implement a number of querries.
