About Fusion Tables Project

CA349 coursework is based on exploiting Fusion Tables to display a large volume of data in a graph or map format.

We had to use a dataset that is suited to a map based layout and we also had to use Fusion Tables to both display the data and allow for queries to filter the dataset visually.

Our front end was to be designed for User ease of use and to allow the user to implement a number of querries.
